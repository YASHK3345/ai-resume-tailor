# 📄 AI Resume Tailor

🚀 An AI-powered web app that tailors your resume and cover letter to any job description in seconds.

🔗 **Live Demo**: [Streamlit App](https://ai-resume-tailor-hjfyr4cs8p3nfk4rtkj2pn.streamlit.app/)  
💻 **Source Code**: This repository

---

## ✨ Features

- 📤 Upload your resume (PDF or DOCX)  
- 📝 Paste any job description  
- 🤖 AI generates a **tailored resume + cover letter**  
- 📥 Download as a professional **PDF**  
- 🌐 Deployed on **Streamlit Cloud**

## ⚙️ Tech Stack

- **Python 3.11**  
- [Streamlit](https://streamlit.io/) – UI framework  
- [OpenAI API](https://platform.openai.com/) – AI resume tailoring  
- [PyPDF2](https://pypi.org/project/PyPDF2/) – PDF parsing  
- [python-docx](https://pypi.org/project/python-docx/) – DOCX parsing  
- [FPDF](https://pypi.org/project/fpdf2/) – PDF generation
